Build the natural language parsing module that translates user route descriptions ("shady waterfront trail", "hilly forest run") into scoring weight adjustments that feed into the existing RouteScorer pipeline.

Purpose: ROUTE-05 is Stride's key differentiator -- users describe vibes, not parameters. The NL parser bridges human intent and algorithmic scoring by converting free-text descriptions into the structured weight format the scorer already understands.

Output: ClaudeClient adapter, prompt templates with JSON schema, NLParser class, and mergeWeights function. All tested with mocked Claude API responses.
</objective>

    **Step 2: Create src/nl/claude-client.js**
    Export class `ClaudeClient`:
    - Constructor takes `apiKey` string. Store it. Set `this.endpoint = 'https://api.anthropic.com/v1/messages'`.
    - Method `async complete(systemPrompt, userMessage, outputSchema = null, options = {})`:
      - Build request body: `{ model: options.model || 'claude-haiku-4-5-20250514', max_tokens: options.maxTokens || 1024, system: systemPrompt, messages: [{ role: 'user', content: userMessage }] }`
      - If outputSchema is provided, add `output_config: { format: { type: 'json_schema', schema: outputSchema } }` to body
      - Send POST to `this.endpoint` with headers:
        - `'x-api-key': this.apiKey`
        - `'anthropic-version': '2023-06-01'`
        - `'content-type': 'application/json'`
        - `'anthropic-dangerous-direct-browser-access': 'true'`
      - If response not ok: parse error body, throw `Error('Claude API error: {status} - {message}')`
      - Parse response JSON. Extract `data.content[0].text`.
      - If outputSchema was provided: `return JSON.parse(text)`
      - Else: `return text`

    **Step 3: Create src/nl/nl-parser.js**
    Import ClaudeClient, NL_PARSER_SYSTEM_PROMPT, WEIGHT_SCHEMA, and mergeWeights (from weights.js).

    Export class `NLParser`:
    - Constructor takes `claudeClient` (a ClaudeClient instance)
    - Method `async parse(userText)`:
      - If userText is falsy (null, undefined, empty string), return null
      - Wrap entire body in try/catch. On any error, log warning and return null (graceful degradation)
      - Call `this.claudeClient.complete(NL_PARSER_SYSTEM_PROMPT, userText, WEIGHT_SCHEMA, { maxTokens: 512 })`
      - Validate result has weights, preferences, vibeKeywords
      - Clamp each weight value to [0, 1] range
      - Return the parsed result object: `{ weights, preferences, vibeKeywords }`

    **Step 4: Add mergeWeights to src/scoring/weights.js**
    Add a new exported function `mergeWeights(regionWeights, nlWeights)`:
    - If nlWeights is null, undefined, or empty object, return `{ ...regionWeights }`
    - Create merged = `{ ...regionWeights }`
    - For each key in nlWeights: if the key exists in merged AND value is a number, override: `merged[key] = nlWeights[key]`
    - Normalize: calculate sum of all merged values, divide each by sum so total = 1.0
    - Return the normalized merged weights
