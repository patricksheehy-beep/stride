  <behavior>
    EngineManager:
    - new EngineManager(orsAdapter, osrmAdapter) stores both adapters in order [ors, osrm]
    - route(waypoints) tries ORS first; if ORS succeeds, returns { route: geojson, engine: 'ors' }
    - route(waypoints) when ORS throws, tries OSRM; if OSRM succeeds, returns { route: geojson, engine: 'osrm' }
    - route(waypoints) when both fail, throws Error('All routing engines failed')
    - route(waypoints) emits 'routing:started' event before routing
    - route(waypoints) emits 'routing:completed' event with { engine, distance } on success
    - route(waypoints) emits 'routing:fallback' event with { from: 'ors', to: 'osrm', reason } when falling back
    - route(waypoints) checks cache before routing; returns cached result if available
    - route(waypoints) caches successful results
  </behavior>
  <action>
    1. Create src/routing/engine-manager.js:
       - Import { eventBus } from '../core/event-bus.js'
       - Import { getCached, setCache } from '../core/cache.js'
       - Export class EngineManager:
         - constructor(orsAdapter, osrmAdapter):
           - this.engines = [ { name: 'ors', adapter: orsAdapter }, { name: 'osrm', adapter: osrmAdapter } ]
         - _buildCacheKey(waypoints): returns `route:${waypoints.map(p => `${p.lat},${p.lng}`).join(';')}`
         - async route(waypoints):
           - Check cache first:
             - const cacheKey = this._buildCacheKey(waypoints)
             - const cached = await getCached('routes', cacheKey)
             - if (cached) { eventBus.emit('routing:cache-hit', { key: cacheKey }); return cached; }
           - eventBus.emit('routing:started', { waypoints })
           - let lastError = null;
           - for (const engine of this.engines):
             - try:
               - const result = await engine.adapter.route(waypoints)
               - const routeResult = { route: result, engine: engine.name }
               - await setCache('routes', cacheKey, routeResult)
               - eventBus.emit('routing:completed', { engine: engine.name })
               - return routeResult
             - catch (err):
               - lastError = err
               - const fromEngine = engine.name
               - const nextIdx = this.engines.indexOf(engine) + 1
               - if (nextIdx < this.engines.length):
                 - eventBus.emit('routing:fallback', { from: fromEngine, to: this.engines[nextIdx].name, reason: err.message })
               - continue
           - throw new Error('All routing engines failed')
