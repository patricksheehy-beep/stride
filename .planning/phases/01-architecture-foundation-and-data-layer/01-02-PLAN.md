    Overpass Adapter:
    - new OverpassAdapter(config) creates adapter with config.overpassEndpoint
    - normalizeOverpassToGeoJSON with a way element returns FeatureCollection with one Feature having type 'LineString'
    - normalizeOverpassToGeoJSON with a way element sets coordinates as [lon, lat] pairs from element.geometry
    - normalizeOverpassToGeoJSON with a way element copies element.tags into Feature properties plus id and osmType:'way'
    - normalizeOverpassToGeoJSON with a relation containing member ways extracts each member as a separate Feature
    - normalizeOverpassToGeoJSON relation member properties include relationId, relationName, routeType, network, osmType:'relation_member'
    - normalizeOverpassToGeoJSON returns { type: 'FeatureCollection', features: [...] }
    - normalizeOverpassToGeoJSON ignores elements without geometry (nodes, incomplete ways)
  </behavior>
  <action>
    1. Create src/data/adapters/overpass.js:
       - Import { buildTrailQuery } from '../query-builder.js'
       - Import { config } from '../../core/config.js'
       - Import { getCached, setCache } from '../../core/cache.js'
       - Import { eventBus } from '../../core/event-bus.js'

       - Export function normalizeOverpassToGeoJSON(overpassData):
         - Initialize features = []
         - Loop over overpassData.elements:
           - If element.type === 'way' && element.geometry:
             - Push Feature with geometry: { type: 'LineString', coordinates: element.geometry.map(n => [n.lon, n.lat]) }
             - Properties: { id: element.id, osmType: 'way', ...element.tags }
           - If element.type === 'relation' && element.members:
             - Loop over members where member.type === 'way' && member.geometry:
               - Push Feature with geometry: { type: 'LineString', coordinates: member.geometry.map(n => [n.lon, n.lat]) }
               - Properties: { id: member.ref, osmType: 'relation_member', relationId: element.id, relationName: element.tags?.name, routeType: element.tags?.route, network: element.tags?.network, ...member.tags }
         - Return { type: 'FeatureCollection', features }

       - Export class OverpassAdapter:
         - constructor(): this.endpoint = config.overpassEndpoint, this.fallbackEndpoint = config.overpassFallbackEndpoint
         - async fetchTrails(bbox, options = {}):
           - cacheKey = `trails:${bbox.join(',')}`
           - Check cache: const cached = await getCached('trails', cacheKey); if (cached) { eventBus.emit('data:cache-hit', { key: cacheKey }); return cached; }
           - Build query: const query = buildTrailQuery(bbox, options)
           - Try primary endpoint, catch and try fallback:
             - const response = await fetch(this.endpoint, { method: 'POST', body: `data=${encodeURIComponent(query)}`, headers: { 'Content-Type': 'application/x-www-form-urlencoded' } })
             - If !response.ok, throw new Error(`Overpass error: ${response.status}`)
           - Parse: const data = await response.json()
           - Normalize: const geojson = normalizeOverpassToGeoJSON(data)
           - Cache result: await setCache('trails', cacheKey, geojson)
           - Emit: eventBus.emit('data:trails-loaded', { featureCount: geojson.features.length, bbox })
           - Return geojson
