    3. Create src/map/route-renderer.js with these exports:

       a. `extractTurnPoints(routeGeoJSON)`:
          - Extract turn-by-turn instruction points from ORS response
          - Navigate to features[0].properties.segments[0].steps
          - For each step, get the coordinate at way_points[0] index from features[0].geometry.coordinates
          - Return array of { lat, lng, ele, instruction, distance, type } objects
          - Return empty array if segments/steps don't exist

       b. `renderRoute(map, routeGeoJSON, options = {})`:
          - Clear any existing route layer group (store reference on map object: map._strideRouteGroup)
          - Create L.layerGroup() and add to map
          - Draw route polyline from coordinates with: color '#E8C547' (gold), weight 4, opacity 0.9
          - Fit map bounds to route with padding [50, 50]
          - Return the layer group

       c. `addDistanceMarkers(map, routeFeature, intervalKm = 1)`:
          - Import @turf/along and @turf/length
          - Calculate total route length in km
          - For each km interval from intervalKm up to (but not including) totalKm:
            - Use turf along() to get point at that distance
            - Create L.marker with L.divIcon: className 'distance-marker', html `<span>${km}</span>`, iconSize [24, 24]
          - Add all markers to the existing route layer group (map._strideRouteGroup)
          - Return array of markers

       d. `addTurnMarkers(map, turnPoints)`:
          - Input: array from extractTurnPoints()
          - For each turn point, create L.circleMarker at [lat, lng]:
            - radius 6, color '#E8C547', fillColor '#0A0A0A', fillOpacity 1, weight 2
            - Bind tooltip with instruction text
          - Add to map._strideRouteGroup
          - Return array of markers

       e. `clearRoute(map)`:
          - Remove map._strideRouteGroup if exists
          - Set map._strideRouteGroup = null

    4. Create tests/map/route-renderer.test.js:
       - Test extractTurnPoints with mock ORS GeoJSON containing segments.steps with way_points
       - Test addDistanceMarkers logic (mock the turf imports, verify correct km intervals calculated)
       - Use mock objects for map and L (no real Leaflet in tests)
