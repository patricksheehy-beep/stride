    2. Create src/map/elevation-chart.js with three exports:

       a. `calculateElevationStats(coordinates)`:
          - Input: array of [lng, lat, ele] coordinate arrays
          - Iterate through coordinates, tracking:
            - totalAscent: sum of positive elevation differences between consecutive points
            - totalDescent: sum of absolute negative elevation differences
            - minEle: minimum elevation seen
            - maxEle: maximum elevation seen
          - Skip coordinates where ele (index 2) is undefined
          - Return { totalAscent, totalDescent, minEle, maxEle, elevationRange: maxEle - minEle }
          - Edge cases: single coordinate returns zeros; no valid elevations returns { totalAscent: 0, totalDescent: 0, minEle: 0, maxEle: 0, elevationRange: 0 }

       b. `renderElevationChart(canvasElement, coordinates)`:
          - Input: DOM canvas element (not ID string), array of [lng, lat, ele] coordinates
          - Calculate cumulative distance between consecutive points using haversine approximation: `Math.sqrt(dlat^2 + dlng^2) * 111.32` km (sufficient for consecutive GPS points)
          - Build Chart.js line chart with:
            - labels: cumulative distance in km (1 decimal)
            - data: elevation values in meters
            - fill: true with backgroundColor 'rgba(232, 197, 71, 0.15)' (gold accent transparent)
            - borderColor: '#E8C547' (gold accent from design system)
            - borderWidth: 2, pointRadius: 0, tension: 0.3
            - responsive: true, maintainAspectRatio: false
            - legend hidden
            - x-axis: title "Distance (km)", tick color '#888'
            - y-axis: title "Elevation (m)", tick color '#888'
            - Grid line color 'rgba(255,255,255,0.1)' for dark theme compatibility
          - Import only needed Chart.js components to allow tree-shaking: `import { Chart, LineController, LineElement, PointElement, LinearScale, CategoryScale, Filler, Tooltip } from 'chart.js'` and register them.
          - Return the Chart instance
          - Downsample coordinates to max 200 points for performance (if > 200, take every Nth point)

       c. `destroyElevationChart(chartInstance)`:
          - Safely destroy a Chart.js instance (null check, call .destroy())
