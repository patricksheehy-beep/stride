       Method: _calculateBbox(startPoint, distanceKm):
         Calculate a bounding box centered on startPoint that's large enough to contain a loop of the requested distance. Use radius = distanceKm / (2 * Math.PI) * 1.5 (50% buffer). Convert to [south, west, north, east] by subtracting/adding latitude and longitude offsets. Latitude offset = radius / 111.32. Longitude offset = radius / (111.32 * cos(lat * pi/180)).

       Method: async generate(startPoint, distanceKm, options = {}):
         1. Emit 'route:generation-started' with { startPoint, distanceKm }
         2. Set store.isGenerating = true
         3. Check cache: key = `generated:${startPoint.lat},${startPoint.lng}:${distanceKm}`. If cached, set store state, emit complete, return cached.
         4. Detect region via detectRegion(startPoint.lat, startPoint.lng). Get weights via getWeightsForRegion(region). Create scorer if region differs from constructor's.
         5. Calculate bbox via _calculateBbox. Fetch trail data via this.overpassAdapter.fetchTrails(bbox). Store in store.trails.
         6. Generate candidates:
            a. Round trip candidates: await this.routeBuilder.generateCandidatesViaRoundTrip(startPoint, distanceKm, 3). Wrap in try/catch.
            b. Waypoint candidate: await this.routeBuilder.generateCandidateViaWaypoints(startPoint, trailData, distanceKm). Wrap in try/catch.
            c. Collect all successful candidates into a single array.
         7. If no candidates, throw new Error('No route candidates could be generated').
         8. Score and rank: call this.scorer.scoreAndRank(candidates, trailData, startPoint).
         9. Add distanceKm (measured via @turf/length) to each ranked result.
         10. Build result: { routes: rankedResults, bestRoute: rankedResults[0] }
         11. Cache result with setCache(ROUTE_STORE, cacheKey, result).
         12. Update state: store.currentRoute = result.bestRoute. store.isGenerating = false.
         13. Emit 'route:generation-complete' with result.
         14. Return result.

         Wrap entire method body in try/catch. On error: store.isGenerating = false, emit 'route:generation-failed' with { error: err.message }, re-throw.
